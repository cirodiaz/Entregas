Instrucciones para ejecutar el programa:

Prerrequisitos:

a. Asegurarse de usar la versión de java 1.8.0_121
b. Crear el archivo "archivoin.txt" en la carpeta "C:\tarea4", o generar un archivo txt de la cantidad de líneas que se desee, con el siguiente formato:

nombreEtiqueta;valor

Donde cada par de nombre de etiqueta y valor serán una línea del archivo; se puede ingresar tantos valores como se desee. Se incluyen dos archivos en la carpeta  /test de la presente entrega.
Las etiquetas pueden ser el nombre que se desee, mientras no se incluya el caracter ";". Los números son de tipo "float". Si se ingresan valores diferentes a números, el programa fallará. 

c. Descargar la carpeta Tarea 4 desde el repositorio git, ya sea usando el comando $ git clone https://github.com/cirodiaz/Entregas.git a una carpeta de su elección

Para hacer uso del programa de tarea 4, por favor realizar el siguiente proceso:

1. abrir la línea de comandos
2. navegar hacia la carpeta "t4_rangos" donde hayan descargado desde el git
3. digitar el comando "mvn package"
4. verificar que el archivo compila correctamente.
5. situarse en la carpeta /target que está en el proyecto maven, y ejecutar el comando java para probar el programa:
java -jar t4_rangos-jar-with-dependencies.jar
6. Seguir las instrucciones del programa.
